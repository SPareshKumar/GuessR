# 🔍 GuessR

GuessR is a modern semantic word-guessing game. Unlike traditional word games that focus on letters (like Wordle), ContextClue challenges your understanding of **semantic similarity**. 

Every guess is ranked based on how close its meaning is to the secret target word using Natural Language Processing.

## ✨ Features

- **Semantic Scoring:** Real-time similarity calculation (0-100%) via API Ninjas.
- **Visual Feedback:** Color-coded progress bars (Blue → Yellow → Green) to visualize your proximity.
- **Hint System:** Progressive hints including word length and the first letter.
- **Clean UI:** A compact, monochromatic blue design with smooth React animations.
- **Dynamic Word Bank:** 250+ curated concrete nouns for diverse gameplay.

## 🛠️ Tech Stack

- **Frontend:** React (Create React App)
- **Styling:** Tailwind CSS v3
- **Icons:** React Icons (Hi2, Bi sets)
- **API:** API Ninjas (Text Similarity)
